Problem 4

Are the following true or false? Assume A is a generic n-by-n matrix.
(a) A^ (-1) equals 1/A
(b) A.^ (-1) equals 1./A

Solution :

(a) False. The operation 1/A is not a valid operation until A is 1x1 matrix.
(b) True. The operation A.^ (-1) raises each entry of A to the power of -1 and the operation 1./A inverts each element of A. So both the operations give same output, hence A.^ (-1) = 1./A.
