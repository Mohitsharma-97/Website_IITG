The plots are differing from each other because when we evaluate the polynomial p in expanded form a lot of multiplication and additon are performed as compared to evaluating the polynomial p in its closed form expression . Due to rounding of numbers done in MATLAB while peforming operations, we don't get exact values of p(x), we kind of only get approximate values of p(x) and this fact is reflected in the above plot when we are trying to plot values of p(x) in closed form and in expanded form. The closed form used 9 multplication operations and 1 substraction operation wheres as the expanded form uses a lot addition and multiplication.

function y = Horner(p,x)
% Evaluates a polynomial p(z)=p_1*z^n+p_2*z^(n-1)+...+p_n*z+p_(n+1)
% at z=x by taking x in real numbers and a vector p with p(i)=p_i, i=1:n+1
% as input.
y=0;
for i=1:(length(p)-1)
y=x.*(p(i)+y);
end
y=p(length(p))+y;
end

function [x, n] = bisect1(p,x_0,x_1,tol)
%Finds a root of a polynomial p(x)=p_1*z^n+p_2*z^(n-1)+...+p_n*z+p_(n+1)
% (provided in terms of a vector p with p(i)==p_i) in an
% interval [x_0,x_1](where p(x_1)p(x_2) < 0) up to a given tolerance tol
% using the Bisection method. We use Horner function to evaluate p(x) at a
% point x.
if tol > x_1-x_0
disp("The tolerance value is greater than the length interval [x_0,x_1]" + ...
". Returning value 0.")
disp("Please input the value of the tolerance " + ...
"less than the length of the interval [x_0,x_1] for sucessful iteration.")
end
if sign(Horner(p,x_0))==sign(Horner(p,x_1))
disp("sign(p(x_0)) and sign(p(x_1)) are same, please enter x_0 and " + ...
"x_1 such that sign(p(x_0)) and sign(p(x_1)) are opposite.")
end
m=0;
iter=0;
x=[];
while (m-2^(1/4)) > tol
iter=iter+1;
m= x_0 + (x_1-x_0)/2;
if sign(Horner(p,x_0))==sign(Horner(p,m))
x_0=m;
else
x_1=m;
end
x=[x m];
end
x=x';
n=iter;
end
